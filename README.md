# DeeVisible

**DeeVisible** is a math-based mobile game developed in Unity for Android.

## Description

DeeVisible is a fast-paced educational game where the player must decide whether the number displayed at the top is divisible by the number displayed at the bottom.

The player swipes right or left depending on the correct answer. As the game progresses, the difficulty increases, making the player react faster and think more carefully.

## Gameplay

The main goal of the game is to test and improve the player's mental math skills.

The player is shown two numbers:

- the top number
- the bottom number

The player must quickly decide if the top number can be divided evenly by the bottom number.

As the game continues, the challenges become harder.

## Controls

- **Swipe Right** – choose one answer
- **Swipe Left** – choose the opposite answer
- **Menu** – navigate through the game menus
- **Options** – adjust game settings

## How to Run

This project is designed for Android.

To run the game:

1. Build the project for Android using Unity.
2. Install the generated APK on an Android device.
3. Open the game and start playing.
